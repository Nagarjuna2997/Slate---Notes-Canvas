## About

Slate is a minimal notes and canvas app for iOS. Combine rich text editing with an infinite canvas to capture everything — from quick notes to detailed sketches — all in one place.

## Features

### Notebooks
- Organize notes into notebooks with custom icons (Health, Work, Personal, Ideas, Games)

### Smart Blocks
- Type `/` to insert blocks: text, headings, lists, tables, charts, math equations (LaTeX), images, audio, files, and canvas

### Infinite Canvas
- Sketch freely on an infinite canvas
- Built-in templates: Blank, Grid, Dotted, Lined
- Full Apple Pencil support
- Favorite your canvases for quick access

### Appearance
- Light and dark mode
- Pure black OLED dark theme

### Export
- Plain Text
- Markdown
- PDF
- Word (.rtf)

Slate does not collect any personal data. All your notes and canvases are stored locally on your device. No analytics, no ads, no third-party tracking.
